# Домашнее задание к занятию "`Резервное копирование`" - `Бурыкин Кирилл`

### Задание 1 Резервное копирование

1.1.Необходимо восстанавливать данные в полном объёме за предыдущий день.

  Подходит полное резервное копирование (full backup), которое делается каждый день. Это позволяет восстановить всю базу данных на конец прошедших суток

1.2.Необходимо восстанавливать данные за час до предполагаемой поломки.

  Нужно полное + инкрементное/дифференциальное копирование

  · Полная копия (например раз в день)

  · Инкрементные копии (каждый час) — сохраняют только изменения с последней резервной копии

  Это позволит восстановить данные: взять последнюю полную копию и применить все инкрементные копии до нужного часа
  
### Задание 2 PostgreSQL


2.1.С помощью официальной документации приведите пример команды резервирования данных и восстановления БД (pgdump/pgrestore).

  Создать копию: pg_dump mydb > backup.sql

  Вернуть базу: psql mydb < backup.sql

2.1.* Возможно ли автоматизировать этот процесс? Если да, то как?

  Да, через cron (планировщик в Linux) 
  Каждый день в 2 ночи делать бэкап: 0 2 * * * pg_dump mydb > /backups/backup_$(date +\%d).sql
